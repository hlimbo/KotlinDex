# KotlinDex
PokeDex App written in Kotlin using Android Jetpack components

## Android Dependencies
* [LiveData](https://developer.android.com/topic/libraries/architecture/livedata) 
* [Navigation](https://developer.android.com/guide/navigation/navigation-getting-started)
* [Paging](https://developer.android.com/topic/libraries/architecture/paging)
* [ViewModel](https://developer.android.com/topic/libraries/architecture/viewmodel)

## 3rd Party Dependencies
* [Coroutines](https://kotlinlang.org/docs/tutorials/coroutines/coroutines-basic-jvm.html)
* [GSON](https://github.com/google/gson)
* [Dagger](https://dagger.dev/)
* [Picasso](https://square.github.io/picasso/)
* [Retrofit](https://square.github.io/retrofit/)
* [Jake Wharton's Coroutine Adapter for Retrofit](https://github.com/JakeWharton/retrofit2-kotlin-coroutines-adapter)
